WebNotes

WebNotes is a free light weight notes tool that creates and saves notes in browser storage. The notes created in WebNotes are auto saved and are only locally available in same browser in same device.

Features:
  • Create and save notes in Markdown syntax.
  • WebNotes uses cookies to make notes persistent and available even if device reboot/shutdown.
  • Notes are auto saved in case of tab/broswer get closed.

Built using HTML, Javascript, Tailwindcss and Less ❤️
